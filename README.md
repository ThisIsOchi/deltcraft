# 🏰 DeltCraft - Medieval Survival Minecraft Server

**DeltCraft** adalah server Minecraft bertema *survival abad pertengahan* buatan komunitas Indonesia. Di sini, pemain diajak menjelajahi dunia penuh misteri, membangun kerajaan megah, menjalin aliansi, hingga bertarung dalam dungeon berbahaya. Semua dirancang untuk menghadirkan pengalaman bermain yang imersif dan seru — baik untuk pemain solo maupun komunitas.

---

## ✨ Fitur Unggulan

- 🏞️ **Survival Bertema Medieval**  
  Kumpulkan sumber daya, bangun kastil, dan ciptakan peradabanmu sendiri.

- ⚔️ **Dungeon & Pertarungan Epik**  
  Hadapi monster kuno, jelajahi reruntuhan, dan raih harta legendaris.

- 🛡️ **Sistem Guild & Aliansi**  
  Bentuk kelompok, kuasai wilayah, dan menangkan perang antar guild.

- 🎉 **Event Mingguan & Musiman**  
  Dari turnamen PvP hingga misi misteri dan minigame.

- 💬 **Komunitas Aktif & Supportif**  
  Tersedia Discord, admin aktif, serta forum pemain untuk berbagi.

- 🏛️ **Cocok untuk Roleplay**  
  Kembangkan karaktermu, buat cerita sendiri, dan bangun dunia bersama.
